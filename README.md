# Sales Data Warehouse

## Project Overview
This project demonstrates a Data Warehouse pipeline built using Python, Pandas, SQLite, and SQL.

The pipeline:
- Extracts sales data from CSV files
- Transforms and enriches the data
- Loads data into a SQLite warehouse
- Executes SQL queries for business reporting
